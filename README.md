# llm-telegram-bot

Проект для ролика -

Node - `v22.18.0`

Используем модель `qwen2.5-coder:14b` для вайб-кодинга

Начальный промпт:

```text
Задача: Предложи архитектуру для Telegram-бота с интеграцией Ollama LLM
Технические требования:

Язык: TypeScript
ORM: Prisma ORM для работы с базой данных
База данных: PostgreSQL/SQLite (на выбор)
Telegram API: node-telegram-bot-api или telegraf
LLM: Интеграция с локальными моделями Ollama

Функциональные требования:

Аутентификация: Пользователи активируют доступ через секретное слово
LLM интеграция: Переадресация сообщений пользователей к Ollama моделям и возврат ответов
База данных: Сохранение информации о пользователях и статусе активации

Что нужно предоставить:

Структуру проекта с папками и основными файлами
Prisma схему для моделей User и других необходимых сущностей
Архитектурную диаграмму основных компонентов
Основные TypeScript интерфейсы для типизации
Примеры ключевых функций:

Обработка команды активации
Отправка запроса к Ollama
Сохранение данных пользователя через Prisma

package.json с необходимыми зависимостями
```
